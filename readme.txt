Allow you to reorder, rename, admin menu items.

== Description ==
Admin Menu Optimizer allows you to reorder and rename the admin menu items in the admin menu. You can easily reorder the menu items by simple drag and drop. You can also rename the menu items to whatever you like.
== Key Features ==
* Reorder admin menu items
* Rename admin menu items
* Drag and drop interface
* Easy to use
* Lightweight
* Clean code

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/admin-menu-optimizer` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
